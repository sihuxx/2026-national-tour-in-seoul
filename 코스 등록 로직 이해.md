## 초기 데이터 로딩

```javascript
const datas = await fetch("/asset/station.json").then(res => res.json()).then(({ data }) => data);
```
- 서버에서 `station.json` 파일을 가져옴
- `.then(res => res.json())` → 받아온 데이터를 JSON으로 변환
- `.then(({ data }) => data)` → JSON 안의 `data` 키만 꺼냄
- 결과적으로 `datas`는 역 목록 배열

## `drawMap()` — 지도 그리기

```javascript
ctx.clearRect(0, 0, canvas.width, canvas.height)
```
- 캔버스 전체를 하얗게 지움. 다시 그리기 전에 항상 호출

### 라인 그리기
```javascript
datas.map((station) => {
  const route = station.route;
  const [x, y] = station.coordinates;
```
- 모든 역을 순회하면서 현재 역의 좌표를 `x, y`로 꺼냄

```javascript
  Object.entries(route).forEach(([key, value]) => {
    value.forEach(valueRoutes => {
      const [toX, toY] = datas.find(stat => stat.idx === valueRoutes).coordinates;
```
- 이 역에 연결된 다음 역들을 순회
- 다음 역의 idx로 `datas`에서 찾아서 좌표를 꺼냄

```javascript
      ctx.beginPath();
      ctx.moveTo(x, y);
      ctx.lineTo(toX, toY);
      ctx.strokeStyle = colorMap[key];
      ctx.stroke();
```
- 현재 역 → 다음 역까지 선을 그림
- 색은 호선(`key`)에 맞는 색으로

### 역 원 그리기
```javascript
datas.map(d => {
  const [x, y] = d.coordinates;
  ctx.beginPath();
  ctx.arc(x, y, 10, 0, Math.PI * 2);
  ctx.fillStyle = colorMap[Object.keys(d.route)[0]];
  ctx.fill();
})
```
- 각 역 좌표에 반지름 10짜리 원을 그림
- `Object.keys(d.route)[0]` → 이 역의 첫 번째 호선 번호로 색 결정

### 역 이름 그리기
```javascript
datas.map(d => {
  const [x, y] = d.coordinates;
  ctx.fillStyle = '#333'
  ctx.fillText(d.name, x + 15, y + 15)
})
```
- 역 좌표에서 오른쪽 아래로 15px 띄어서 역 이름 텍스트 표시

## `drawRoute(course)` — 선택한 경로 하이라이트

```javascript
drawMap()
```
- 먼저 지도를 깨끗하게 다시 그림

```javascript
const stationList = course.path.map(idx => findStationByIdx(idx))
```
- 경로의 idx 배열을 실제 역 객체 배열로 변환

### 환승역 파악
```javascript
course.path.forEach((idx, i) => {
  if (i === 0 || i === course.path.length - 1) return
```
- 출발역, 도착역은 환승 체크 안 함

```javascript
  const prevEdge = (map[course.path[i - 1]] || []).find(e => e.to === idx)
  const nextEdge = (map[idx] || []).find(e => e.to === course.path[i + 1])
```
- 이 역으로 들어오는 엣지(이전→현재)와 나가는 엣지(현재→다음)를 찾음

```javascript
  if (prevEdge && nextEdge && prevEdge.line !== nextEdge.line) {
    transferSet.add(idx)
  }
```
- 들어온 호선이랑 나가는 호선이 다르면 → 환승역으로 판단해서 `transferSet`에 추가

### 노란 경로 라인
```javascript
ctx.beginPath()
ctx.strokeStyle = "yellow"
ctx.lineWidth = 5
stationList.forEach(({ coordinates: [x,y] }, i) => {
  i === 0 ? ctx.moveTo(x, y) : ctx.lineTo(x, y)
})
ctx.stroke()
```
- 경로 순서대로 첫 역은 `moveTo`(펜 올리기), 나머지는 `lineTo`(선 긋기)
- 다 그린 후 `stroke()`로 실제로 선 표시

### 마커 그리기
```javascript
stationList.forEach((station, i) => {
  const isStart = i === 0
  const isEnd = i === stationList.length - 1
  const isTransfer = transferSet.has(station.idx)

  ctx.arc(x, y, isTransfer ? 12 : 10, 0, Math.PI * 2)
  ctx.fillStyle = isStart ? "blue" : isEnd ? "red" : isTransfer ? "orange" : "yellow"
  ctx.fill()
})
```
- 출발역 → 파랑
- 도착역 → 빨강
- 환승역 → 주황 (크기도 12로 더 크게)
- 일반역 → 노랑

## `getDistance()` — 두 역 사이 거리 계산

```javascript
const [x1, y1] = stationA.coordinates
const [x2, y2] = stationB.coordinates
return Math.round(Math.sqrt(Math.pow(x2 - x1, 2) + Math.pow(y2 - y1, 2)))
```
- 피타고라스 공식: √((x2-x1)² + (y2-y1)²)
- `Math.round`로 소수점 버리고 정수로 반환

## `map` 구성 — 그래프 만들기

```javascript
datas.forEach(station => {
  const currentIdx = station.idx
  map[currentIdx] = []
```
- 각 역의 idx를 키로 해서 빈 배열 초기화

```javascript
  Object.entries(station.route).forEach(([line, nextStationArr]) => {
    nextStationArr.forEach(nextIdx => {
      const dist = getDistance(station, findStationByIdx(nextIdx))
      map[currentIdx].push({ to: nextIdx, line, distance: dist })
    })
  })
})
```
- 이 역에서 갈 수 있는 모든 다음 역을 순회
- `{ to: 다음역idx, line: 호선, distance: 거리 }` 형태로 저장
- 결과: `map[역idx] = [{ to, line, distance }, ...]`

## `getTop5Paths()` — 경로 탐색 (DFS)

```javascript
const dfs = (current, visited, currentPath, currentLine, totalDistance, transfers) => {
```
- `current`: 현재 있는 역
- `visited`: 이미 방문한 역 목록
- `currentPath`: 지금까지 지나온 경로
- `currentLine`: 현재 타고 있는 호선
- `totalDistance`: 지금까지 이동한 총 거리
- `transfers`: 지금까지 환승한 횟수

```javascript
  if (visited.has(current)) return;
```
- 이미 방문한 역이면 중단 → 무한루프 방지

```javascript
  if (current == endNode) {
    paths.push({ path: fullPath, routeText: ..., distance: totalDistance, transfers })
    return
  }
```
- 도착역에 도달하면 경로를 `paths`에 저장하고 종료

```javascript
  const nextVisited = new Set(visited)
  nextVisited.add(current)
```
- 현재 역을 방문 처리한 새 Set 만들기 (원본 안 건드림)

```javascript
  neighbors.forEach(({ to, line, distance }) => {
    const isTransfer = currentLine !== null && currentLine !== line
    const nextTransfers = isTransfer ? transfers + 1 : transfers
    dfs(to, nextVisited, [...currentPath, current], line, totalDistance + distance, nextTransfers)
  })
```
- 연결된 모든 다음 역으로 재귀 호출
- 호선이 바뀌면 환승 횟수 +1

```javascript
return paths.sort((a, b) => a.distance - b.distance).slice(0, 5)
```
- 찾은 모든 경로를 거리 짧은 순으로 정렬 후 5개만 반환

## `renderCourseList()` — 코스 목록 렌더링

```javascript
const sorted = [...paths].sort((a, b) =>
  sort === 0 ? a.distance - b.distance :
  sort === 1 ? a.transfers - b.transfers : 0
)
```
- `sort === 0` → 거리 짧은 순
- `sort === 1` → 환승 적은 순

```javascript
list.innerHTML = sorted.map(...).join("")
```
- 정렬된 경로들을 HTML로 변환해서 한번에 렌더링

```javascript
$$(".course-list .item").forEach((item, i) => {
  item.onclick = () => {
    $$(".course-list .item").forEach(el => el.classList.remove("selected"))
    item.classList.add("selected")
    selectedCourse = sorted[i]
    drawRoute(selectedCourse)
  }
})
```
- 각 목록 항목 클릭 시:
  1. 모든 항목에서 `selected` 클래스 제거
  2. 클릭한 항목에만 `selected` 추가 (하이라이트)
  3. 클릭한 경로를 `selectedCourse`에 저장
  4. 지도에 해당 경로 표시

## 버튼 이벤트들

### 검색 버튼
```javascript
$(".search-btn").onclick = () => {
  if (!startPlace || !endPlace) {
    alert("출발지와 도착지를 모두 설정해주세요")
    return
  }
  resultPath = getTop5Paths(startPlace, endPlace)
  renderCourseList(resultPath, 1)
}
```
- 출발/도착 중 하나라도 없으면 알림 후 종료
- 둘 다 있으면 경로 탐색 후 환승 적은 순(1)으로 목록 렌더링

### 정렬 버튼들
```javascript
sortBtns.forEach((btn, i) => {
  btn.onclick = () => {
    if (resultPath.length === 0) { alert("코스를 먼저 지정해주세요"); return }
    renderCourseList(resultPath, i)
    sortBtns.forEach(allBtns => allBtns.classList.remove('active'))
    btn.classList.add("active")
  }
})
```
- 검색 결과 없으면 알림
- 버튼 index(`i`)가 sort 기준이 됨 (0=거리, 1=환승)
- 클릭한 버튼만 `active` 클래스 추가

### 일정설정 버튼
```javascript
$(".schedule-btn").onclick = () => {
  if (!selectedCourse) { alert("선택된 투어 코스가 없습니다"); return }
  localStorage.setItem("course", JSON.stringify(selectedCourse))
  $(".page-course").style.transform = 'translateX(-100%)'
  setTimeout(() => { location.href = "/schedule" }, 500)
}
```
- 선택된 코스 없으면 알림 후 종료
- `localStorage`에 선택한 코스 저장 (다음 페이지에서 꺼내 씀)
- 현재 페이지를 왼쪽으로 슬라이드
- 0.5초 후 `/schedule` 페이지로 이동
